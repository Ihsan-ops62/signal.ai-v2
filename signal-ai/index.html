Connect LinkedIn

📋 How to get your LinkedIn Access Token

1
Go to LinkedIn Developer Apps and create a new app (or use an existing one).
2
In your app, go to Products and request access to "Share on LinkedIn" and "Sign In with LinkedIn using OpenID Connect".
3
Go to Auth tab → OAuth 2.0 scopes. Make sure openid, profile, w_member_social are enabled.
4
Use the OAuth 2.0 Token Generator under Auth tab to generate a token, OR use a tool like oauth.tools with your Client ID & Secret.
5
Paste the access_token (starts with AQV... or similar) below.
⚠️ Tokens expire after ~60 days. You'll need to refresh it when it expires.

Connect Facebook

📋 How to get Facebook Page Access Token

1
Go to Facebook Developers and create an app (Business type).
2
Add the Facebook Login and Pages API products.
3
In Graph API Explorer, select your app and generate a Page Access Token (choose your page).
4
Copy the token and your Page ID (found in your page's About section).
⚠️ Token expires after ~60 days. You can extend it using the debug_token endpoint.
